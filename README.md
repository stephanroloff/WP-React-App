# WP REACT APP

###Steps<br><br>
1.Clone<br>

```bash
git clone https://github.com/stephanroloff/wp-react-app.git<br><br>
```

2.Move to the folder<br>

```bash
cd wp-react-app/app<br><br>
```

3.Install Dependencies<br>

```bash
npm install<br><br>
```

4.Run for Development:<br>

```bash
npm run dev<br><br>
```

and for production:<br>

```bash
npm run build<br><br>
```

5.That's all!<br>
